Политика конфиденциальности

Последнее обновление: 20 июня 2026 г.

1. Общие сведения

ChronoFlow — это Progressive Web App (PWA) для отслеживания времени и событий, разработанная dituals758 при поддержке AI-ассистента MiMo Code (Xiaomi MiMo Team).

Данная Политика конфиденциальности описывает, как приложение обрабатывает данные пользователей.

2. Сбор данных

ChronoFlow НЕ собирает, не хранит и не передаёт никакие данные на внешние серверы.

  • Все события и настройки хранятся исключительно локально на устройстве пользователя (localStorage).
  • Нет регистрации, нет аккаунтов, нет серверной части.
  • Нет аналитики, нет трекеров, нет cookie.
  • Нет доступа к интернету для передачи данных.

3. Service Worker и кэширование

Приложение использует Service Worker для:

  • Кэширования файлов приложения (HTML, CSS, JS, иконки) для офлайн-работы.
  • Обслуживания приложений из локального кэша при отсутствии интернета.

Service Worker не отправляет данные на сторонние серверы и не отслеживает активность пользователя.

4. Данные, хранящиеся на устройстве

Приложение хранит следующие данные в localStorage:

  • События — название и дата/время созданного события.
  • Настройки — тема оформления, видимость часов.

Эти данные покидают устройство только при использовании функции «Экспорт», инициированной пользователем вручную.

5. Функция «Поделиться» и импорт

При использовании функций импорта/экспорта данных файлы обрабатываются полностью на устройстве. Данные не передаются по сети.

6.third-party сервисы

Приложение не использует сторонние SDK, библиотеки аналитики, рекламные модули или сервисы мониторинга.

7. Дети

Приложение не собирает персональные данные и не имеет возрастных ограничений. Оно безопасно для использования детьми.

8. Безопасность

Данные хранятся в защищённом хранилище браузера. Приложение не реализует передачу данных по сети, что исключает риски перехвата.

9. Изменения в политике

В случае изменений в Политике конфиденциальности обновлённая версия будет опубликована в этом документе с указанием даты обновления.
